# EasyWebshop integration

EasyWebshop (opens new window) is hosted e-commerce platform. They will run the shop infrastructure for you for a monthly fee.

WARNING

Please be aware that this integration is not maintained by the BTCPay Server team. If you have any questions, please contact EasyWebshop (opens new window) directly.
